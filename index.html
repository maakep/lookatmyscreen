Look At My Screen

Peer-to-peer screen sharing. No server. No internet. Just a link.

Share Your Screen

Start sharing, then generate invite codes for each viewer. Each viewer gets their own code and sends back a response to connect.

  1. Click Start Sharing to pick a screen/window
  2. Send each viewer's invite code to them
  3. Paste each viewer's response code and click Connect
  4. Use + Add Viewer for additional viewers

View a Screen

Paste the sharer's invite code to generate a response code. Send it back to complete the connection.

  1. Paste the sharer's invite code below
  2. Click Generate Response
  3. Copy the response code and send it back to the sharer
  4. Once connected, the shared screen takes over the page
