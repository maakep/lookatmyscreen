Look At My Screen

Peer-to-peer screen sharing. No backend. Just a link.

Share Your Screen

Start sharing, then send the invite link to each viewer. The viewer will send you a response link back. Open it, and the connection completes automatically.

  1. Click Start Sharing to pick a screen/window
  2. Send each viewer's invite link (via Slack, email, etc.)
  3. The viewer sends you a response link back - just click it
  4. Use + Add Viewer for additional viewers

View a Screen (Manual)

If you received a raw code instead of a link, paste it here to connect manually.

  1. Paste the sharer's invite code below
  2. Click Generate Response
  3. Copy the response code and send it back to the sharer
